## 🛠️ 컨벤션
### `Branch`
- 개발중인기능/#이슈번호

### `Commit`
- `feat`: 새로운 기능
- `fix`: 버그 수정
- `refactor`: 리팩토링
- `design`: 사용자 UI 디자인 변경
- `comment`: 필요한 주석 추가 및 변경
- `docs`: 문서 관련 (추가, 수정, 삭제, README)
- `test`: 테스트
- `chore`: 기타 변경사항 (assets, 라이브러리 추가, 패키지 매니저 등)
- `init`: 초기 생성
- `rename`: 파일/폴더명 수정, 이동
- `remove`: 파일 삭제
